# OpenClaw WSL Launcher

这是一款专为 Windows 用户量身打造的 **OpenClaw WSL 桌面图形化启动器与高级会话管理工具**。通过一个直观的 GUI 界面，它让你无需在终端中敲击繁琐的命令行，就能轻松管理运行在 WSL (Windows Subsystem for Linux) 中的 OpenClaw 网关。

## 🌟 项目背景与痛点解决

**OpenClaw** 是一个强大的大语言模型代理网关，但当它被部署在 Windows 的 WSL 环境中时，日常使用往往伴随着以下痛点：

1. **终端依赖症**：每次想启动、停止或查看网关状态，都需要打开终端，输入 `wsl`，然后再输入 `openclaw` 的各种命令，流程繁琐。
2. **“僵尸进程”困扰**：网关卡死或端口被占用时，小白用户往往束手无策，只能重启整个电脑。
3. **会话历史“滚雪球”**：
   - 在**本地 WebChat (仪表盘)** 聊天时，多开的测试会话容易堆积，导致界面杂乱。
   - 在接入**飞书 (Feishu) 机器人**时，由于大模型会不断读取旧的历史上下文，随着对话越来越长，机器人回复会变得极度缓慢甚至卡死。
4. **定位飞书“罪魁祸首”难**：当飞书变卡时，由于底层的日志和会话文件都是以哈希 ID 命名的（如 `82aa6ede...`），你很难知道到底是哪个群聊或私聊拖慢了系统。

**本项目就是为了彻底解决上述痛点而生的！**

## 🚀 核心功能

### 1. 通用网关运维 (小白友好)
- **一键启停与重启**：点一下按钮即可启动 (`Start`) 或停止 (`Stop`) 网关。新增的 `Restart` 按钮能强制清理僵尸进程并重启服务，专治各种“不服”。
- **可视化配置**：内置 **基础配置 (Settings)** 弹窗，小白也能直接在界面上修改 WSL 发行版名称（如 `Ubuntu-24.04`）和监听端口，配置自动持久化保存。
- **状态与健康监控**：一键查看网关当前状态、端口监听情况和系统健康指标。
- **安全退出快照**：收工时点击 `Safe Exit`，自动保存当前运行状态快照并安全停机，下次启动时会在界面上提示上次退出的时间。

### 2. 双模式高级会话管理
启动器创新性地将操作分为两大场景：

#### 🌐 WebChat / 仪表盘模式（针对本地开发与测试）
- **一键 Launch**：启动网关 + 自动在 Windows 浏览器中打开带 Token 的 Dashboard。
- **清理单会话 (Prune)**：一键清空冗余的测试会话，只保留干净的主会话。
- **重置与归档 (Reset / Archive)**：当当前对话太长时，一键切断历史上下文，让大模型“重新做人”，告别卡顿。

#### 💬 Feishu / 飞书模式（针对机器人运维）
- **智能会话识别 (Detect Feishu)**：自动扫描底层杂乱的 JSONL 文件，提取最近用户和机器人的聊天内容，生成“人话摘要”，让你一眼认出“这是刚才那个股票群的聊天”。
- **会话软关闭 (Close Selected)**：怀疑某个长对话拖慢了系统？选中它，点击关闭，该会话会被从活动索引中移除，但原始聊天记录依然安全保留在硬盘上。
- **深度归档 (Archive Selected)**：确定要彻底摆脱某段历史？一键将其移动到备份目录，彻底释放大模型上下文压力。

## 🎯 使用场景

- **日常开发启动**：每天早上来到工位，双击启动器，点击 `Launch`，网关启动、浏览器弹出，直接开始和本地模型对话。
- **飞书卡顿急救**：同事反馈飞书机器人回复太慢。你打开启动器切换到 `Feishu 模式`，点击 `Detect`，发现第一名是一个长达几百条记录的群聊。你选中它点击 `Archive`，机器人瞬间恢复如初。
- **修改底层配置**：换了新的 API Key，只需要点击右上角的 `Settings` 确认端口，然后点击 `Restart`，新配置立刻生效。
- **下班收工**：点击 `Safe Exit`，服务优雅停止，安心关机。

- **AI 部署“小白”用户**：只会照着教程安装环境，但对 Linux 命令不熟悉，极度依赖图形化界面进行日常操作。

## 🛠️ 快速上手

1. **准备工作**：确保你的 Windows 电脑已安装 WSL2，并在其中（默认推荐 `Ubuntu-24.04`）正确部署了 OpenClaw 环境及 Python 3。
2. **下载项目**：将本项目克隆或下载并解压到 Windows 本地任意目录。
3. **首次配置**：如果你使用的不是 `Ubuntu-24.04` 或默认的 `18789` 端口，双击运行 `OpenClaw-WSL-Launcher.bat`，点击右上角的 **[基础配置 / Settings]** 修改并保存。
4. **开始使用**：根据你当前的场景选择 **WebChat** 或 **Feishu** 模式，点击对应的按钮即可！

> 💡 **进阶提示**：你可以直接把 `OpenClaw-WSL-Launcher.bat` 发送到 Windows 桌面作为快捷方式，并为它换上一个好看的图标！
